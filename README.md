# Telegra.ph Rust Bot

Цей бот автоматизує створення публікацій на платформі [Telegra.ph](https://telegra.ph/) за допомогою Rust та Selenium. Він підтримує роботу з базою даних SQLite та CSV-файлами.

## Можливості

- **Автоматична публікація**: Створення постів через браузер Chrome (Selenium).
- **Збереження в БД**: Кожна успішна публікація зберігається в SQLite.
- **Експорт у CSV**: Можливість вивантажити всі записи з бази даних у CSV-файл.
- **Публікація з CSV**: Масове створення постів на основі даних із CSV-файлу.

## Попередні вимоги

Для роботи програми вам потрібно встановити:

1.  **Rust**: [Встановити Rust](https://www.rust-lang.org/tools/install).
2.  **Google Chrome**: Має бути встановлений у системі.
3.  **ChromeDriver**: Версія має відповідати вашій версії Chrome.
    - Ви можете завантажити його з [офіційного сайту](https://googlechromelabs.github.io/chrome-for-testing/).

## Налаштування та запуск

### 1. Запуск ChromeDriver

Програма очікує, що ChromeDriver запущений на порту `9515`. Відкрийте окремий термінал і виконайте:

```bash
chromedriver --port=9515
```

### 2. Підготовка бази даних

Програма автоматично створить папку `database/` та файл бази даних при першому запуску, але переконайтеся, що у вас є права на запис у директорію проекту.

### 3. Запуск бота

Виконайте команду:

```bash
cargo run
```

## Використання (main.rs)

У файлі `src/main.rs` ви можете керувати режимами роботи бота, розкоментовуючи потрібні рядки:

```rust
// 1. Створити одну тестову публікацію
// service.publication().await?;

// 2. Експортувати всі дані з SQLite в CSV
service.db_to_csv("db_export.csv").await?;

// 3. Прочитати CSV та створити нові пости на Telegra.ph
service.csv_to_telegraph("db_export.csv").await?;
```

## Структура проекту

- `src/main.rs` — точка входу, керування логікою.
- `src/service.rs` — логіка роботи з Selenium (TelegraphService).
- `src/repository.rs` — робота з базою даних SQLite.
- `src/model.rs` — структура даних `Publication`.
- `database/database.db` — файл бази даних.
- `publication.csv` — логі за замовчуванням.

## Компіляція (Збірка проекту)

Ви можете зібрати готовий виконуваний файл для своєї операційної системи.

### Для Linux

1. Виконайте команду збірки:
   ```bash
   cargo build --release
   ```
2. Виконуваний файл з'явиться за шляхом: `target/release/telegra_bot`
3. Для запуску скопіюйте цей файл разом із папкою `database/` (або створіть її поруч).

### Для Windows

1. Переконайтеся, що встановлено Rust (версія MSVC або GNU).
2. Виконайте команду:
   ```powershell
   cargo build --release
   ```
3. Виконуваний файл з'явиться за шляхом: `target\release\telegra_bot.exe`
4. Для роботи на іншому ПК вам також знадобляться:
   - `telegra_bot.exe`
   - `database/` (папка з БД)
   - `chromedriver.exe` (має бути запущений)

## Кросс-компіляція (З Linux на Windows)

Якщо ви на Linux і хочете зібрати файл для Windows, найпростіше використати `mingw`:

1. Встановіть таргет: `rustup target add x86_64-pc-windows-gnu`
2. Зберіть проект:
   ```bash
   cargo build --release --target x86_64-pc-windows-gnu
   ```

## Поширені помилки

- **WebDriver Error**: Переконайтеся, що `chromedriver` запущений і порт `9515` вільний.
- **Database Locked**: Закрийте інші програми, які можуть використовувати `database.db`.
- **CSV Error**: Переконайтеся, що CSV-файл не відкритий в іншій програмі (наприклад, Excel) під час запису.
