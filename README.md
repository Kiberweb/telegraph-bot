# Telegraph Automation Bot

Цей проект дозволяє автоматизувати публікацію статтей на [Telegra.ph](https://telegra.ph) за допомогою PHP та Symfony Panther. Бот підтримує роботу з базою даних SQLite для логування та CSV-файлами для імпорту/експорту.

## 🚀 Основні можливості
- **Публікація через браузер**: Автоматизація введення заголовка, автора та контенту.
- **Імпорт з CSV**: Масова публікація статтей, прочитаних з файлу.
- **Логування в DB**: Всі опубліковані пости зберігаються в SQLite з отриманими посиланнями.
- **Експорт в CSV**: Можливість вивантажити всі записи з бази назад у CSV.
- **Транслітерація**: Генерація латинських посилань (slug) для українських текстів.

## 🛠 Вимоги
- PHP 8.2 або вище
- Composer
- Google Chrome
- Chromedriver (має відповідати версії вашого Chrome)

## 📦 Встановлення

1. Клонуйте репозиторій.
2. Встановіть залежності:
   ```bash
   composer install
   ```
3. Створіть необхідні папки:
   ```bash
   mkdir -p data database
   ```

## 🏃‍♂️ Запуск

1. Запустіть Chromedriver у окремому терміналі:
   ```bash
   chromedriver --port=9515
   ```

2. Запустіть основний скрипт:
   ```bash
   php index.php
   ```

## 📁 Структура файлів
- `src/Entity/Article.php` — модель статті та логіка транслітерації.
- `src/Service/TelegraphService.php` — сервіс автоматизації браузера.
- `src/Repository/` — робота з SQLite та CSV.
- `data/articles.csv` — файл для імпорту статтей.
- `database/database.sqlite` — база даних з логами публікацій.

## 📝 Формат CSV для імпорту
Для коректного імпорту файл `data/articles.csv` повинен мати такі заголовки:
`Title,Author,Content,Url,Created At`

Приклад рядка:
`"Мій пост","Автор","Текст контенту","",""`
